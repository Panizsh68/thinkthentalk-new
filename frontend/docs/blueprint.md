# **App Name**: Think Then Talk Community

## Core Features:

- Project Setup: Sets up the basic Next.js project with TypeScript, Tailwind CSS, ESLint, and Prettier configured for consistent code style.
- React Query Configuration: Adds React Query to the project and sets up a basic QueryClientProvider at the root of the app for efficient data fetching.
- Tailwind CSS Configuration: Configures Tailwind CSS to scan all relevant files (pages/app, components, etc.) to ensure all styles are included.
- Placeholder Page: Initializes a simple placeholder page indicating that the project setup is successful ('Event Platform Frontend – Setup OK').
- UI State management setup: Includes a state solution for managing UI state of components, such as user interface status and state

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) for a sophisticated and modern feel.
- Background color: Light Lavender (#E6E6FA) as a desaturated version of indigo.
- Accent color: Orchid (#DA70D6) for highlights and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, machined, objective look.
- Clean and minimal layout to ensure ease of use and focus on event content.
- Subtle transitions and animations for a smooth user experience when navigating between pages.
